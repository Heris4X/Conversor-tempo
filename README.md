# ⏳ Conversor de Unidades de Tempo

Projeto de software web estático desenvolvido de forma individual para a disciplina da Universidade Estadual da Paraíba (UEPB).

## 👤 Autor (Projeto Solo)
* **Herikles Silva Santos** - GitHub: [@Heris4X](https://github.com/Heris4X)

## 📌 Sobre o Projeto
O sistema permite a conversão rápida e intuitiva entre diversas unidades de tempo (Milissegundos, Segundos, Minutos, Horas, Dias e Semanas) de forma 100% client-side.

## 🔄 Histórico de Versões (Releases)

### 📦 v1.0.0 - Versão Funcional (Foco em Lógica)
* **Objetivo:** Garantir o funcionamento matemático de todas as conversões.
* **Estética:** Extremamente simples e crua (serif/Times New Roman), focando apenas na entrega das funcionalidades básicas exigidas.

### 🎨 v2.0.0 - Versão Estética (Foco em UI/UX)
* **Objetivo:** Transformar a interface em um produto moderno, atraente e responsivo.
* **Estética:** Layout moderno com fundo em gradiente, sombras suaves, transições ao passar o mouse, cantos arredondados, fontes limpas (Inter) e adaptável para qualquer tamanho de tela.
